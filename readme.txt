@anywhere for Wordpress is a Wordpress plugin lets you easily integrate Twitter's @anywhere platform with your Wordpress blog.

Instructions:

Part 1: Register an @anywhere application

1. Go to http://dev.twitter.com/anywhere/apps/new
2. Fill in all the fields.Be sure to set your URL accurately. You may leave the "Callback URL" field blank.
3. Accept the terms
4. Click "Edit Application Settings"
5. Click "Application Detail"
6. Copy and paste your "Consumer Key" somewhere you won't lose it. You'll need this later.

Part 2: Install the plugin as you would any other Wordpress plugin.
